# Cursor Extension for Metanoia-QA

Enables AI-powered QA automation within Cursor IDE.

## Setup

1. Copy `metanoia-mcp.md` to `.cursor/mcp.json`
2. Update the path to point to your Metanoia-QA installation
3. Restart Cursor

## Features

- Inline test status display
- One-click sprint start
- Real-time agent monitoring
- Test case autocomplete
