# Metanoia-QA IDE Extensions

## VS Code Extension

Install from `.vsix` file or package yourself:

```bash
cd extensions/vscode
npm install
npm run compile
npm run package
code --install-extension metanoia-qa-*.vsix
```

## Cursor MCP Integration

See `cursor/metanoia-mcp.md` for configuration.

## Features

- Inline test status indicators
- One-click sprint management
- Real-time agent monitoring
- Test case autocomplete
